# Flight Booking & Travel Data Analysis

## Project Overview
This project presents an interactive Power BI dashboard analyzing flight booking and travel data.  
It focuses on understanding passenger behavior, revenue trends, travel patterns, and payment methods.

## Objectives
• Analyze flight booking trends  
• Understand revenue generation from ticket sales  
• Identify passenger travel patterns  
• Compare airline and city-wise performance  
• Deliver insights using interactive visualization  

## Tools & Technologies

• Power BI  

## Power BI Dashboard Features

• Booking Analysis (total bookings & trends)  
• Revenue Insights (ticket price analysis)  
• Passenger Age Analysis  
• Travel Class Distribution  
• Payment Method Analysis  
• Airline-wise Revenue Comparison  
• City-wise Ticket Price Analysis  
• Interactive filters and slicers  

## Dataset

The dataset includes:

• Booking ID  
• Passenger Age  
• Travel Class  
• Ticket Price  
• Payment Method  
• Airline  
• Departure City  
• Distance (km)  
• Flight Duration (hours)  

## Key Insights

• Total bookings reached around 4000  
• Revenue generated is approximately $10M+  
• Economy and First class have high booking counts  
• Net Banking and UPI are commonly used payment methods  
• Certain cities generate higher ticket revenue  
• Passenger age group between 25–50 dominates bookings
